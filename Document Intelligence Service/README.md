## ğŸ”§ Configuration

### OCR Settings
Edit `config/__init__.py`:
```python
# OCR Settings
OCR_LANGUAGE = 'vi'  # Vietnamese
OCR_DETECTION = True
OCR_RECOGNITION = True

# Upload Settings
```

### AI Settings
Edit `.env`:
```bash

GEMINI_API_KEY=your_api_key_here
ENABLE_AI_ENHANCEMENT=True

# AI Feature Flags

ENABLE_EXTRACTION=True
ENABLE_SUMMARY=True
ENABLE_QA=True
ENABLE_TRANSLATION=True
```
