# 1. Download a small dataset (20-50 images)
# 2. Place images here with .txt caption files
# 3. Run: quickstart.bat
# 4. Choose "Small dataset" configuration
# 5. Train for 2-3 epochs to test

# 1. Copy small_dataset_config.yaml to my_test_config.yaml
# 2. Edit train_data_dir to point to this folder
# 3. Set num_train_epochs to 2
# 4. Run: train.bat configs\my_test_config.yaml
