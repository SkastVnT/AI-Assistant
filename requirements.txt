# QUICK START
# ============================================================================
#
# 1. Clone repository:
#    git clone https://github.com/SkastVnT/AI-Assistant.git
#    cd AI-Assistant
#
# 2. Choose a service to setup:
#    - cd ChatBot              (for chatbot service)
#    - cd "Text2SQL Services"  (for SQL conversion)
#    - cd "Speech2Text Services" (for transcription)
#    - cd stable-diffusion-webui (for image generation)
#
# 3. Follow service-specific README.md for detailed setup
#
# 4. Each service runs independently with its own:
#    - Virtual environment (venv_*)
#    - requirements.txt
#    - Configuration (.env)
#    - Port number
#
