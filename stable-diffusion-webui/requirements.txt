# INSTALLATION GUIDE

# 
# 1. Create virtual environment:
#    python -m venv venv_sd
#    
# 2. Activate virtual environment:
#    Windows: .\venv_sd\Scripts\activate
#    Linux/Mac: source venv_sd/bin/activate
#
# 3. Install PyTorch with CUDA:
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
# 4. Install other dependencies:
#    pip install -r requirements.txt
#
# 5. Download Stable Diffusion models:
#    Place .safetensors or .ckpt files in models/Stable-diffusion/
#
# 6. Run WebUI:
#    python webui.py --api

# 7. For ChatBot integration, use --api flag
